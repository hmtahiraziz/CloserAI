# Objection handling

**Price** — Acknowledge, clarify scope, share approved package ranges, offer ROI framing in demo.

**Timing** — Offer a short discovery now; schedule callback if mid-crisis.

**Authority** — Ask who else evaluates; offer to include them in the meeting.

**Competitor** — Ask what works/doesn’t; position complementary strengths.

**Security** — Share privacy summary; offer specialist for deep security review.

Never argue. Confirm whether the concern was resolved.
